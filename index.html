部署在 Vercel 上时，默认使用同域代理（/api），无需额外配置。如需自定义代理地址可在此修改。

📖 关于 CORS 和代理

由于浏览器安全策略（CORS），网页无法直接调用 api.anthropic.com。本站通过 Vercel Serverless Function 代理 API 请求，同域名无跨域问题。

如果你自行部署（如 GitHub Pages），需要设置自己的代理。推荐方案：

  • Vercel（推荐）：Fork 本项目后在 Vercel 导入，自动拥有代理
  • Cloudflare Worker：创建 Worker 粘贴代理代码，将 URL 填入上方
  • 自建服务器：将 /api/v1/messages 请求转发到 api.anthropic.com
